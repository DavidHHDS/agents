# 5. Diagrama de Secuencia (Teoría)

### Teoría
Ilustra el orden cronológico estricto de los mensajes entre clases, actores o componentes para realizar una operación específica.

### Interacciones
* **Síncrono (`->`):** Emisor bloqueado a la espera de respuesta.
* **Retorno (`-->`):** Respuesta a un mensaje previo.
* **Asíncrono (`->>`):** Disparar sin esperar respuesta continua.
* **Fragmentos (`alt`, `opt`, `loop`):** Lógica condicional, bucles y lógica alternativa integrada en el flujo.

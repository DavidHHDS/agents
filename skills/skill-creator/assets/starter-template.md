# Plantilla Base para Nueva Skill

Copia esta estructura para iniciar una nueva skill compatible con la especificación.

### 1. SKILL.md
```markdown
---
name: [nombre-de-la-skill]
description: [Explicación de qué hace y cuándo usarla]
---

# [Título de la Skill]

[Instrucciones generales e índice de navegación]

## Recursos Adicionales
- Consulta la [guía de referencia](references/reference.md) para detalles técnicos.
- Revisa los [ejemplos](assets/examples.md) para ver implementaciones.
```

### 2. Estructura de Carpetas
```text
skill-name/
├── SKILL.md
├── references/
│   └── reference.md
├── assets/
│   └── examples.md
└── scripts/
```
